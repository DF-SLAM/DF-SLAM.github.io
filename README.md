本地查看：
1. 在 VS Code 插件市场搜索并安装 Live Server。
2. 在 VS Code 中打开 website-ccode\DF-SLAM.github.io 这个文件夹。
3. 点击 VS Code 窗口右下角状态栏的 Go Live 按钮。
4. 浏览器会自动打开 http://127.0.0.1:5500/index.html，此时 Viser 播放器就能完美在本地预览和交互了。
